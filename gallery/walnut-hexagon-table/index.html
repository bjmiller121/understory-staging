Hexagon Boardgame Table

This walnut table is an interesting deviation from my other rectangular boardgame tables. While sharing many design elements and the same functionality of my other tables, this hexagon shape is ideal for fitting a large playing surface in a smaller footprint.

Round tapered legs and an arched apron give the table a light and elegant stance while the inlayed maple striping in the border bring some visual interest when the table is open for gaming mode.

This table features an accessory attachment system for cupholders as well as some new card holders that I prototyped as a little bonus feature for the clients.

Materials: Walnut, felt

Dimensions: 66" Wide by 54" Deep by 30" Tall
