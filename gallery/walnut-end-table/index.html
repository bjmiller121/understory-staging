End Table with Drawer

This solid walnut end table borrows design elements from my walnut coffee table. With clean lines and a light appearance, this Shaker-inspired table fits well with many aesthetics. The maple drawer on solid wood slides seamlessly blends in with the apron when closed.

Materials: Black Walnut, Maple (drawer box)

Dimensions: 14" Wide by 24" Deep by 22" Tall
