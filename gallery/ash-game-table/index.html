Ash Boardgame Table

This boardgaming table is designed to serve as a traditional dining table when not in use for tabletop games. It can simply be converted from dining table to gaming table by removing the four leaves, revealing the recessed and felt-lined play surface below.

A lot of time and consideration went into designing a table that has the functionality of a gaming table with a recessed play area while maintaining a light and clean aesthetic. My aim was for it to pass as just a dining room table if you didn't know better. This meant packing all the structure and functionality in as small a vertical space as possible to get a comfortable table height and ample leg room underneath.

Materials: White ash, felt (play surface liner)

Dimensions: 78" Wide by 54" Deep by 30" Tall
